# Survival Of The Fattest

**Survival of the Fattest** is a singleplayer game where you control a hungry panda. Devour other animals, consume the environment, and become the fattest to survive.

## Controls
- Move - Mouse Hover
- Sprint - Left Click
- Menu - TAB/ESC

*This game was made in 1 weekend for the 2026 Kenney Jam.*
